# Wrapped 2025

A cinematic, scroll-based narrative experience about discovering a passion for computer science and building.

## Stack

- React + Vite
- Tailwind CSS
- Framer Motion
- React Bits components

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Navigation

- Scroll or use arrow keys to navigate between slides
- Click the slide indicators on the right to jump to specific slides
